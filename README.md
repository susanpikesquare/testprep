# PrepQuest

Native iOS test prep app for GED / High School Equivalency study.

The app is designed for a learner who feels discouraged, has trouble sustaining attention, and needs small, believable wins. The product goal is not just practice questions. It is to rebuild confidence through short skill-building sessions that make progress visible.

## First Version

- Native iOS app for iPhone first, with iPad support.
- Built with SwiftUI.
- Local-first progress tracking.
- Short practice sessions of 3 to 5 questions.
- Calm, non-shaming feedback.
- Skill map for Math, Language Arts, Science, and Social Studies.
- Review mode for missed questions.
- A simple daily "one step" flow.

## Design Principles

- Start easy enough that the first session creates momentum.
- Reward returning, trying, and improving, not only correct answers.
- Keep screens uncluttered and choices limited.
- Use plain language explanations.
- Avoid timers until Nick has enough confidence for test simulation.
- Treat missed answers as information, not failure.
